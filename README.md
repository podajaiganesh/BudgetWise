# BudgetWise
> Track smarter. Spend better.

## About the Project

BudgetWise is a personal finance and budget planning application that helps users track their income and expenses, monitor spending habits, and stay within their monthly budget.

The application is designed to provide meaningful financial insights through data visualization, budget monitoring, and spending analysis instead of simply recording transactions. By helping users understand where their money goes, BudgetWise encourages better financial decisions and more effective budget management.

This project is being developed as a full-stack web application using Java, React, and MySQL while following modern software engineering principles, clean architecture, and industry-standard development practices.

## Problem Statement

Many students and young professionals struggle to keep track of their daily expenses. Transactions are often recorded in notebooks, spreadsheets, or not recorded at all, making it difficult to understand spending patterns or identify unnecessary expenses.

BudgetWise addresses this problem by providing a simple, organized, and visual platform for recording expenses, managing budgets, and understanding financial behavior through reports and analytics.

## Project Vision

The goal of BudgetWise is to build a clean, reliable, and user-friendly budgeting application that helps users make informed financial decisions through clear visualizations, budget alerts, and spending insights.

Rather than functioning as a simple expense tracker, BudgetWise aims to become a practical financial companion that encourages responsible spending habits.

## Planned Features (V1)

- **Authentication** — Registration, Login, JWT-based session handling, User Profile
- **Dashboard** — Income/Expense overview, monthly summary, remaining budget, recent transactions
- **Expense Management** — Add, edit, delete, search, and filter expenses by category or date
- **Income Management** — Add, edit, and delete income entries
- **Budget Planning** — Monthly and category-wise budget limits with remaining budget tracking
- **Analytics** — Expense distribution, monthly spending trends, category comparisons, and calculated spending insights
- **Alerts** — Budget and category overspending warnings
